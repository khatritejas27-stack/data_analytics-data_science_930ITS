# what is protocol explain all list of protocol  ? 

## what is protocol  ??

1. protocol is set of rules i.e called protocol  
2. protocol is used in web technologies 
3. protocol is used to access any web url securely or un securely   

## list of types of protocol 

1. http  (http://www.raviflutes.com/) not secured on server 
2. https (https://www.raviflutes.com/) secured on server
3. MIME  (multipurpose internet mail extension)
4. SMTP  (simple mail transfer protocol)
5. POP   (post office protocol)
6. IP    (IP address) 
7. ftp   (file transfer protocol)
8. TCP   (transmission commision protocol)
